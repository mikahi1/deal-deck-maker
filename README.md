# Deal Deck Maker

A standalone static HTML app for making and printing a custom Deal Deck.

## Render

This repo includes `render.yaml` for a Render Static Site.

- Service type: Static Site
- Build command: `echo "No build required"`
- Publish directory: `.`
- Entry file: `index.html`

After connecting the GitHub repo in Render, Render will publish the site from the root directory and redeploy on each commit.
